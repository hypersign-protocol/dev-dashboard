## Server


### Intallation

```
cd dashboard/server
npm i
```

### Env=production

```sh
npm run build
npm run bootstrap
npm run setupPlan
npm run start
```

Note: make sure you have `production.env` file. 

### Env=development


```sh
npm run bootstrap:dev
npm run setupPlan:dev
npm run dev
```

Note: make sure you have `development.env` file.
